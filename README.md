# VeriDx.ai_Website
=======
# VeriDx.ai marketing site (React + Tailwind)

Professional landing page for **VeriDx.ai** (AI-powered intelligence for medical malpractice evidence review), built with **React (JSX)** and **Tailwind CSS**.

## What’s included

- **Hero**: “The Truth, Diagnosed” + clear positioning
- **Challenge / Approach / Value / Who we work with / Status / Contact** sections
- **Medical-legal styling**: clinical blues/teals, restrained neutrals, modern cards
- **Responsible-use disclaimers**: not diagnosis or legal judgment; informational only
- **Status**: selective early partners; no public demos (per PDF)

## Run locally

```bash
cd veridx-ai
npm install
npm run dev
```

## Build

```bash
cd veridx-ai
npm run build
npm run preview
```

## Update copy / sections

Main page is in `src/App.jsx`. Global styling utilities are in `src/index.css`.
